# Sections Directory

Landing page sections for the FinBiz finance application.
Each section is a self-contained component representing a major part of the landing page.

## Landing Page Sections (in order):

1. **Hero** - Main hero section with headline, CTA, and hero image
2. **Statistics** - Key metrics and data visualization
3. **Features** - "Boosting Business, Today and Tomorrow" section
4. **About** - "Real-time accounting at your fingertips" section
5. **Services** - Service offerings and benefits
6. **Testimonials** - "What are people saying" customer reviews
7. **Pricing** - "Find the right plan" pricing tiers
8. **CallToAction** - Final CTA section before footer
9. **Newsletter** - Email subscription section (optional, can be in Footer)

## Structure:
Each section should be in its own folder:
```
Hero/
  ├── Hero.jsx
  └── index.js
```

## Naming Convention:
- PascalCase for section components
- Descriptive names that match the Figma design
